🌍 World Population Analysis Project

This project focuses on exploring and analyzing global population trends using historical and recent data across countries and continents.

📊 Dataset Features

The dataset contains the following columns:

rank → Country population rank
cca3 → 3-letter country code
country → Country name
continent → Continent name
2023 population → Population in 2023
2022 population → Population in 2022
2020 population → Population in 2020
2015 population → Population in 2015
2010 population → Population in 2010
2000 population → Population in 2000
1990 population → Population in 1990
1980 population → Population in 1980
1970 population → Population in 1970
area (km²) → Total land area in square kilometers
density (km²) → Population density per square kilometer
growth rate → Population growth rate
world percentage → Percentage of world population
🎯 Project Objectives
Analyze population growth trends over time
Compare population across continents
Identify top populated countries
Study population density distribution
Visualize historical population changes
